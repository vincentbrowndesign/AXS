# Ghost Team Review

## Signal
- Is measurement above marketing?
- Does each section tie back to the rep?
- Is the message clear in one pass?

## Decision
- Does the page make the next action obvious?
- Are there too many branches?
- Is the CTA direct?

## Interface
- Does it feel like an instrument?
- Is spacing clean?
- Is hierarchy strong on mobile?

## System
- Can each section become real product logic?
- Is anything fake or decorative only?
- Does the screen connect to actual software behavior?

## Real Time
- Does the page imply live truth, not delayed summary?
- Is capture first and profile second?

## Behavior
- Does it measure action instead of self-description?
- Does it show output instead of motivation?
